# PAP Distribution Repository

Pre-built self-contained executables for 
[PAP](https://packagist.org/packages/pixelbrackets/pap) (PHP App Publication).

## Available Downloads

- `pap-linux-x64` - Self-contained binary for Linux x64 (PHP 8.2 bundled, ~50MB)
- `pap.phar` - Universal PHAR for all platforms (requires system PHP 7.2+, ~27MB)
- `checksums.txt` - SHA256 verification for all files

## Version Matching

Each release version matches the main PAP package version.
